# Branding Prompts

## Positioning

- product name: `Codex Memory Skill`
- core message: `local-first, skill-native, testable memory workflow`
- visual keywords:
  - local-first
  - memory graph
  - routing
  - terminal
  - index
  - structured knowledge
- style keywords:
  - modern open-source branding
  - geometric
  - clean dark neutral background
  - cyan / blue / teal accent
  - precise, technical, trustworthy

## Logo Prompt

Use for `docs/images/logo.png`.

```text
Design a modern open-source software logo for "Codex Memory Skill". 
Create a clean geometric symbol that combines three ideas: local memory, routing, and a coding terminal. 
Use a minimal icon mark made from stacked nodes, a memory chip / archive block, and a subtle terminal angle bracket motif. 
Style: flat vector-like, crisp edges, high contrast, premium technical aesthetic, no mockup, no 3D, no text outside the mark. 
Palette: deep charcoal background with cyan, blue, and teal accents. 
The result should feel like a serious GitHub developer tool, not a gaming brand. 
Center composition, square canvas, clean negative space, sharp silhouette, export-ready logo.
```

## Social Card Prompt

Use for `docs/images/social-card.png`.

```text
Create a GitHub-style open-source project social card for "Codex Memory Skill". 
Theme: local-first memory workflow for coding agents. 
Show a clean technical composition with a central layered diagram: CLI -> route -> asset index -> memory graph. 
Include abstract visual cues for repository files, structured memory notes, routing lines, and local execution. 
Style: elegant developer tooling branding, paper-figure clarity, dark neutral background, cyan / blue / teal highlight colors, subtle glow, precise layout, premium but restrained. 
Make the image look like a serious hero graphic for a top-tier GitHub project README. 
No screenshots of real UIs, no human characters, no excessive text, no clutter. 
16:9 landscape composition with a strong central focal point and clear visual hierarchy.
```
